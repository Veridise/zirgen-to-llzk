Detailed Description

template<typename Typ, typename Factory = NullptrFactory<Typ>>
class zklang::CopyablePointer< Typ, Factory >

A owning smart pointer that when copied creates a new pointer in the copy using the copy constructor of the pointee type.

Default initializes to the result of the factory method and by default the factory returns nullptr.
