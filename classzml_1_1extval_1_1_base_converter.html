Detailed Description

This interface exposes the logic for representing extending finite field elements for a particular field.

Implementations of this interface define how the Ext* family of lowerable ops are converted into lower level operations. Implementations are meant to use LLZK since this lowering is intended to happen in the conversion pass from ZML to LLZK.
