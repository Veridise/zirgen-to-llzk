const zhl::TypeBinding & binding ) const
inline

Add a type binding to a value created during the dialect conversion.

This should be used when a replaceUses* rewriter function is used in a pattern, this way dependants of the previous value can reference a type binding linked to the new value.

The cast is done over an injected object to the pattern so while this breaks constness on the whole pass it doesn't on the pattern itself. In addition, this is an append change so it is safe to do as long as the value has not been inserted already.

Extracts the binding from the input value/operation and creates a cast of the value into the type materialized from the binding.

If the super type is specified generates an additional SuperCoerceOp from the binding's type to the super type. REQUIRES that the operation has one result only.
