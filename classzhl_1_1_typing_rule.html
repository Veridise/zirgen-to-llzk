virtual mlir::FailureOr< Frame > zhl::TypingRule::allocate ( Frame ) const
inlinevirtual

Returns a new inner frame slot in the frame if the operations needs to allocate one.

By default returns failure(), signifying that the op does not need to allocate a frame.

Reimplemented in zhl::BlockTypeRule, zhl::MapTypeRule, zhl::ReduceTypeRule, and zhl::SwitchTypeRule.
