TypeBinding zhl::expr::interpreter::interpretOp ( Op ,
const TypeBinding & binding,
Args && ... )

Interpret an operation and return a new binding with a constant expression if one could be computed or with the expression removed if either the interpreter failed or the op does not support constant expressions.

By default removes any expression that may be in the binding. To handle ops that may return a constant expression create specializations of this template.

Examines the construct op and checks if it could be used to construct an affine expression.

If the constructed component does not match with an semi-affine operator or any of the operands do not have a ConstExpr then gives up an returns the binding without a ConstExpr. A binding that does not have a ConstExpr means that it does not encode a semi-affine expression.
